Показ цен рецептов (предмета, суммы реагентов) в окне профессий.

Измененный мод LilSparkysWorkshop https://github.com/laytya/LilSparkysWorkshop-vanilla v2.0.1 для WoW версии 3.3.5+ сервера Sirus.

  В окне профессии мода Skillet - работает.
  В окне рецептов мода AckisRecipeList (все рецепты, включая неизученные) - работает базово.
    Обновление базы рецептов из открытого окна профессии ARL,
    вызовом функции "/run UpdateRdbSkills()" для каждой профессии отдельно.
  Обновление цен в базе работает, вызовом функции "/run UpdateRdbPrices()", потом будет кнопка.

  Запуск:
  1. Cкопировать папки LilSparkysWorkshop и AckisRecipeList в \Interface\Addons\ (в папке игры).
  2. Открыть окно ARL на нужной профессии, вставить в чат игры сообщение "/run UpdateRdbSkills()"
      и запустить один раз для каждой профессии (окно профессии должно быть открыто в ARL).
  4. Вставить в чат игры сообщение "/run UpdateRdbPrices()" и запустить для обновления цен.
    Цены обновляются все сразу, но только те, которые видил ЛилСпарки, а видит он не все пока.
